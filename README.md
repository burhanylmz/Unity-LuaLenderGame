# Unity-LuaLenderGame

**LuaLender**, klasik arcade oyunlarından ilham alan, Unity motoru ile geliştirilmiş fizik tabanlı bir uzay aracı iniş simülasyonudur.

## 🚀 Özellikler
* **Hassas Fizik Kontrolü:** Aracın ivmesini ve iniş hızını dikkatli yönetmeniz gerekir.
* **Yakıt Yönetimi:** Sınırlı kaynakla en güvenli inişi planlayın.
* **Zorlu Seviyeler:** Farklı yerçekimi ve engel yapılarına sahip platformlar.

## 🛠️ Teknik Detaylar
* **Motor:** Unity 2022+ 
* **Dil:** C#
* **Mekanikler:** Rigidbody2D fiziği, Vector tabanlı itki kuvvetleri.
